## What Continuum gives you

Each turn, Continuum assembles your context from:
- Your identity core (who you are — stable, never compressed)
- Dynamic context retrieved from a 26K-entry corpus of past sessions + project documentation, selected by semantic relevance to the current conversation
- Compressed summaries of older turns in this session
- Full recent conversation history

This means you have access to knowledge from previous sessions that a normal Claude Code session would have forgotten. Use it. Reference it. Build on it.
